# Autoencoder Project

This repository is a simple starting point for experimenting with autoencoders.
The current version only includes the project scaffold and placeholder code.

## Folder Overview

- `main.py`: Entry point for the project.
- `config/`: Configuration values such as batch size and learning rate.
- `data/`: Local data files or dataset placeholders.
- `src/`: Core source files for loading data, defining models, training, testing, and utilities.
- `saved/`: Saved model checkpoints.
- `outputs/`: Generated outputs such as plots or reconstructions.
